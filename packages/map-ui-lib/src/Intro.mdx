# @ogc-maps/storybook-components

Welcome to the **Storybook Maps** component library!

This library provides a set of reusable, config-driven React components for building interactive web maps based on OGC API standards.

## Key Features

- 🧩 **Framework Agnostic**: Pure UI components, no hard dependency on MapLibre GL JS suitable for any React map application.
- ⚙️ **Config Driven**: All components are powered by a strongly-typed JSON configuration (Zod validated).
- 🌍 **OGC Standard**: First-class support for OGC API Features and Tiles (perfect pair with `tipg`).
- 🎨 **Themable**: Built with Tailwind CSS and easily customizable.

## Getting Started

Install the package:

```bash
pnpm add @ogc-maps/storybook-components
```

Import styles in your app root:

```tsx
import '@ogc-maps/storybook-components/dist/style.css';
```

## Core Components

Explore the components in the sidebar:

- **LayerPanel**: Manage layer visibility, opacity, and ordering.
- **Legend**: Display legends for active layers, with simple, categorical, and gradient display modes; optional per-layer opacity slider.
- **BasemapSwitcher**: Switch between different base maps.
- **SearchPanel**: Search and filter layer features by text, number, datetime, and select fields with CQL2 output.
- **FeatureDetailPanel**: Show properties of a selected map feature (panel or modal variant).
- **FeatureTooltip**: Compact property preview tooltip, positioned by the caller.
- **ExportButton**: Trigger CSV export of one or more layers (single button or dropdown).
- **CoordinateDisplay**: Mouse cursor coordinates with cycling format selector (decimal / DMS).
- **CollapsibleControl**: Collapsible icon-button wrapper for placing controls as map overlays.

## Admin Components

Used by the admin UI to build the visual config wizard. Can also be used in custom admin interfaces.

- **StyleEditor**: Visual property editor for fill, line, circle, and symbol styles. Supports symbol mode (text/icon/both), "fx" toggle for data-driven categorical/gradient color expressions, and 70+ MapLibre paint/layout properties with live preview.
- **LegendEditor**: Manage legend entries with a "Generate from Style" button when a style with a color expression is provided.
- **InfoTip**: Accessible CSS-only tooltip icon; auto-used by `FormField`'s `description` prop.
